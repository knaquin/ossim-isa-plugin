Detailed Description

Maintains data and functionality associated with a common point between CMP and REF images.

Given the list of correlation peaks associated with a patch correlation, and the IVT transform for residuals, this object can recompute residuals after an adjustment, and find the most consistent peak (or none at all).

EXPLANATION: REF is the image in which the features are located. A small REF patch (typ. 16x16) is extracted over each feature. The CMP image contains the larger patch in which the matching point (area) is searched. The correlator establishes correlation peaks for each candidate match. The CMP image point is saved in the CorrelationPeak object corresponding to the REF image point saved in this object. Ideally, when all things are registered, the CMP view pt of the feature will be equal to the corresponding REF view pt. Any difference between the two is the residual, defined as residual = CMP_view_pt - REF_view_pt, i.e., the vector originating on the REF image and ending on the CMP image.
