void ISA::CorrelationAtpGenerator::initialize ( )
privatevirtual

Initializes the specific generator.

Should be overriden by derived class as needed. The base class implementation here creates the processing chains with ortho resampling for the inputs. If the derived generator is to work in raw image space, the derived class needs to override. TODO: Seems like processing in ortho-space is specific to the correlation-based ATP generator. Need to move this functionality to CorrelationAtpGenerator child class.
