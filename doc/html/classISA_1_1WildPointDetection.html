int  maxDistance = 10000 
) const

Detects wild points using weighted averages.

Detects wild points using a weighted average. Returns a vector containing booleans whether a point is an outlier or not. A vector is determined to be an outlier based off a weighted average with a activation function defined (linear, gaussian, or exponential).

Parameters
activationDefines activation function used to determine weight
maxDistanceParameter used for activation function. How it is used is dependent on which activation function.

< On/Off weight vector (Default only 100 pixels or less are on)

< Convergence is when no changes to the retVal were made during the loop

< If the vector is turned off or not
