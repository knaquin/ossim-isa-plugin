Detailed Description

Manages all objects associated with correlating two images.

Given an image pair with overlap, this class lays out a set of tie-points areas and establishes correlations and corresponding residuals for each area. NOTE: All references to pixel coordinates in this class are assumed to be in output projection (view) space. The primary product of this class is a list of CorrelationSet objects, each representing a set of correlation peaks for each tie-point area. The TP list provided is populated with the TP locations and correlation sets found.
